# Project Management System

A collaborative project management platform for planning, tracking, and delivering projects efficiently.

## Features

- Project Management
- Kanban Board
- Sprint Planning
- Task Assignment
- Time Tracking
- File Sharing
- Team Collaboration
- Activity Logs
- Dashboard
- Reports

## Tech Stack

React.js

Node.js

Express.js

MySQL

Socket.IO

## Installation

```bash
git clone <repo>

npm install

npm run dev
```
